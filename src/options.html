General options

Show the AdBlock button next to the address bar

To make the button optional, we put him in a helper extension all by himself, to be installed along with AdBlock.

To add the button, click Install on the Browser Button for AdBlock page.

To remove the button, right click him and choose Disable.

I like the text ads on Google search results; show me those. Beta

Subscribe to filter lists

The defaults are great for English-speaking users in the US. Others may wish to subscribe to more lists.

Don't subscribe to more than you need -- every one slows you down a tiny bit! Credits and more lists can be found here.

Sample CSS selectors What they match
div[class="ad"] any div whose class is "ad"
table td img[class^="Ad"][id="bigAd"] any img (within tds within tables) whose class starts with "Ad" and whose id is "bigAd"
iframe[name$="AdFrame"][src*="ads.com"]:last-child any iframe whose name ends with "AdFrame"
and whose src contains "ads.com" and who is the last element in its parent node
Sample domain regexes What they match
.* any domain
mail..*.com any domain containing "mail." and ".com" in that order
mail.google.com any domain containing "mail.google.com"
^mail.google.com$ any domain exactly matching "mail.google.com"

Got a particularly useful filter? Why not share it with us, so we can give it to all AdBlock users?

Shortcut: Press Ctrl-Shift-K (think "Kill!") on a page to block an ad. (If you don't see anything happen, try clicking on the page first.) Shortcut enabled

Don't run AdBlock on domains that end with...

For example, "google.com" whitelists google.com, mail.google.com, and www.google.com.

Shortcut: Press Ctrl-Shift-L (think "Live!") on a page to keep AdBlock from running on that page's domain. Shortcut enabled

For contributors only

These options are only useful when debugging.

If you use one of these and have a bug or ad to report, it would help if you include any debug output in the report. Thanks so much for contributing!

Show debug statements in Console Log (which slows down AdBlock)

Show timing statements in Console Log (which, ironically, slows down AdBlock a tiny bit. Thanks a lot, Heisenberg...)

This is the time between AdBlock receiving the list of filters from the background page and AdBlock's code finishing. It does not include the time Chrome spends rendering the page or parsing new style rules.
